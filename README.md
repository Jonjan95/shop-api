# Shop API

Ett Spring Boot-projekt för att hantera produkter, byggt med Java 21 och MySQL.
Skapat som en del av utbildningen Systemutvecklare (Test).

## Förkrav

* Java 21
* Maven
* MySQL Server

## Installation & Start

### 1. Konfigurera Databasen
Projektet kräver en lokal MySQL-databas.
Logga in i din MySQL-klient och kör:

sql
CREATE DATABASE shopdb;
